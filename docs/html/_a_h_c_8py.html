Detailed Description

Module implementing a Coherent Ising Machine with Amplitude Heterogenuity Correction (CIM-AHC) on GPU.

This module contains functions for simulating and analyzing a Coherent Ising Machine (CIM) with Adaptive Heuristic Correction (AHC) using GPU acceleration. It includes implementations of:

  • Linear time scheduling
  • Custom feedback and pump scheduling
  • Divergence computation
  • Max-Cut energy calculation
  • The main CIM-AHC simulation function
  • Result plotting utilities

The CIM-AHC algorithm is designed to solve Ising model problems and can be applied to various optimization tasks, including Max-Cut problems. The implementation uses PyTorch for GPU acceleration and supports batch processing for parallel simulations.
